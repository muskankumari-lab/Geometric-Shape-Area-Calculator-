# Geometric-Shape-Area-Calculator-
Java OOP project demonstrating abstraction, interfaces, and polymorphism to calculate areas of Circle &amp; Square shapes.

# README

## Project Title

**Geometry Area Calculator**

## Introduction

This is a simple Java mini project that calculates the area of different geometric shapes. The program allows the user to enter dimensions and then displays the area of a square, triangle, and circle.

## Objective

* To calculate the area of basic geometric shapes.
* To understand the use of classes, objects, and methods in Java.
* To practice user input using the Scanner class.

## Features

* Calculates area of a square.
* Calculates area of a triangle.
* Calculates area of a circle.
* Takes input from the user.

## Technologies Used

* Java
* Scanner Class

## OOP Concepts Used

* **Class:** Geometry
* **Object:** Object of Geometry class
* **Methods:** Used for area calculations

## Formulas Used

* Area of Square = side × side
* Area of Triangle = ½ × base × height
* Area of Circle = 3.14 × radius × radius

## How to Run

1. Compile the program:

   bash
   javac Geometry.java
   
2. Run the program:

   bash
   java Geometry
   

## Sample Output


Enter side of square: 5

Enter base of triangle: 10

Enter radius of circle: 7

## Conclusion

This project demonstrates the use of Java methods and object-oriented programming concepts to calculate the area of different geometric shapes. It is simple, easy to understand, and suitable as a beginner-level mini project.
